Custom Backlit Macropad


The custom backlit macropad is my first end-to-end personal hardware project designed to host a keyboard-building event for students on campus.

The PCB was designed from scratch in Altium, manufactured, and reflow soldered by hand. It has eight SMD components which give it a programmable backlighting color. The case is laser cut from various materials.

It consists of four programmable keys with firmware written for QMK compatibility. Standard operation is Previous Track, Pause/Play, Next Track, and Mute.

Steno-Anki Engine


Steno-Anki engine is a CLI tool that efficiently parses, decodes, and creates stenographic briefs to export in a format that Anki natively supports.

Originally built to manage Clippy2 files for Plover, it was later extended to support custom dictionaries, phoneticize results, and use a fine-tuned GPT model to generate custom briefs.

Useful for both beginning and advanced stenographers to help improve speed and reduce the toil of brief creation and memorization.
