Steno-Anki Engine


Steno-Anki engine is a CLI tool that efficiently parses, decodes, and creates stenographic briefs to export in a format that Anki natively supports.

Originally built to manage Clippy2 files for Plover, it was later extended to support custom dictionaries, phoneticize results, and use a fine-tuned GPT model to generate custom briefs.

Useful for both beginning and advanced stenographers to help improve speed and reduce the toil of brief creation and memorization.
